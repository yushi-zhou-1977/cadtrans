一、插件简介
================================================================================

  本插件用于在AutoCAD图纸中翻译越南文文本，也支持英语、日语、韩语
  翻译为中文或英文。

  翻译由 MyMemory 网络翻译服务提供，需要电脑联网。

  免费额度：
    - 不填邮箱：每天 5,000 字符
    - 填写邮箱：每天 50,000 字符（推荐，在设置窗口中填写）

================================================================================
二、安装插件（加载到AutoCAD）
================================================================================

  步骤1：找到插件文件
    文件位置：项目文件夹下的 BG\bin\Release\CadTrans.dll

  步骤2：打开AutoCAD

  步骤3：加载插件，有以下两种方式：

    ┌─────────────────────────────────────────────────────────┐
    │ 方式A：用命令加载（推荐）                                │
    │                                                         │
    │  1) 在AutoCAD底部命令行中输入: NETLOAD                   │
    │  2) 按回车键                                            │
    │  3) 弹出文件选择窗口，找到 CadTrans.dll 文件              │
    │  4) 选中文件，点击"打开"                                 │
    │  5) 加载成功，无报错即可使用                              │
    └─────────────────────────────────────────────────────────┘

    ┌─────────────────────────────────────────────────────────┐
    │ 方式B：让AutoCAD每次启动自动加载                         │
    │                                                         │
    │  1) 先用方式A加载一次插件                                │
    │  2) 在命令行输入: APPLOAD                               │
    │  3) 按回车键                                            │
    │  4) 在弹出的窗口下方找到"启动组"(Startup Suite)           │
    │  5) 点击"内容"(Contents)按钮                            │
    │  6) 点击"添加"(Add)，选择 CadTrans.dll                   │
    │  7) 点击"关闭"，以后每次启动AutoCAD会自动加载             │
    └─────────────────────────────────────────────────────────┘

  注意：
    - 如果弹出安全提示，请选择"加载"或"始终加载"
    - 加载后不需要重复加载，除非关闭AutoCAD重新打开
    - 如果提示"无法加载"，请确认AutoCAD版本为2020或兼容版本

================================================================================
三、首次使用前的设置
================================================================================

  加载插件后，建议先完成以下设置：

  步骤1：在命令行输入: SetTranslateDefaults
  步骤2：按回车键，弹出设置窗口
  步骤3：在"邮箱地址"栏填写你的邮箱（提升每日翻译额度到50,000字符）
  步骤4：根据需要调整其他参数：
         - 翻译文本高度：翻译后中文文字的大小（默认250）
         - 文本偏移距离：中文文字与原越南文之间的距离（默认500）
         - 翻译模式：
           · 新建文本 — 在原文下方创建绿色中文，原文保留（默认）
           · 替换文本 — 用中文直接覆盖原文
  步骤5：点击"保存"按钮

五、命令详细操作步骤
================================================================================

------------------------------------------------------------------------
5.1 快速翻译命令 TranslateVNQuick（最常用，推荐）
------------------------------------------------------------------------

  用途：将图纸中的越南文快速翻译成中文，自动在原文下方创建绿色中文。

  步骤1：在命令行输入: TranslateVNQuick
  步骤2：按回车键
  步骤3：命令行提示"请选择需要翻译的越南文文本:"
  步骤4：用鼠标点击要翻译的文字（可以按住Shift点击选择多个）
         也可以用鼠标框选（从左到右拖动框住文字）
  步骤5：选好后按回车键
  步骤6：等待翻译完成，命令行会显示进度，例如：
         [1/3] 正在翻译: - BÊ TÔNG... -> - 混凝土... [成功]
  步骤7：翻译完成后，命令行显示统计结果

  翻译结果：
    - 原文保留不变
    - 在原文下方出现绿色中文文字
    - 中文文字使用黑体（SimHei）显示

  注意：
    - 翻译过程中CAD会短暂等待，这是正常的（每个文字约1秒）
    - 翻译需要联网，请确保电脑可以访问互联网
    - 只能选择文字对象（单行文字或多行文字），其他对象会被自动过滤

------------------------------------------------------------------------
5.2 完整翻译命令 TranslateVN
------------------------------------------------------------------------

  用途：翻译时可以自定义源语言、目标语言、翻译模式等参数。

  步骤1：在命令行输入: TranslateVN
  步骤2：按回车键
  步骤3：选择源语言，命令行显示：
         选择源语言: [vi/id/en/ja/ko]
         输入对应字母后按回车：
           vi = 越南语（默认）
           id = 印尼语
           en = 英语
           ja = 日语
           ko = 韩语

  步骤4：选择目标语言，命令行显示：
         选择目标语言: [zh-CN/zh-TW/en/id]
         输入对应代码后按回车：
           zh-CN = 简体中文（默认）
           zh-TW = 繁体中文
           en = 英语
           id = 印尼语

  步骤5：选择翻译模式，命令行显示：
         选择翻译模式: [n/r]
           n = 新建文本 — 在原文旁创建翻译（默认）
           r = 替换文本 — 直接替换原文内容

  步骤6：输入翻译文本高度（数字），按回车
         默认值250，直接按回车使用默认值

  步骤7：如果选择了"新建文本"模式，输入偏移距离（数字），按回车
         默认值500，直接按回车使用默认值

  步骤8：用鼠标选择要翻译的文字对象
  步骤9：按回车开始翻译

------------------------------------------------------------------------
5.3 设置翻译参数 SetTranslateDefaults
------------------------------------------------------------------------

  用途：设置邮箱地址、文本高度、偏移距离和翻译模式。

  步骤1：在命令行输入: SetTranslateDefaults
  步骤2：按回车键
  步骤3：根据提示依次输入参数：

         输入邮箱地址（提升额度至50,000字符/天）[当前: 未填写]:
         - 直接按回车：保持当前设置不变
         - 输入邮箱后按回车：提升每日翻译额度到50,000字符
         - 无需注册验证，任意有效邮箱即可

         输入翻译文本高度 [当前: 250]:
         - 控制翻译后中文文字的大小
         - 直接按回车使用默认值250

         输入文本偏移距离 [当前: 500]:
         - 控制中文文字与原越南文之间的距离（仅新建模式有效）
         - 直接按回车使用默认值500

         选择翻译模式 [当前: 新建]:
         - 输入 n 或按回车：新建模式，保留原文，在下方创建绿色中文
         - 输入 r：替换模式，用中文直接覆盖原文

  步骤4：所有参数设置完成后，命令行会显示最终设置结果。

------------------------------------------------------------------------
5.4 输入许可证密钥 SetLicenseKey
------------------------------------------------------------------------

  用途：输入许可证密钥，激活正式版授权。

  步骤1：在命令行输入: SetLicenseKey
  步骤2：按回车键
  步骤3：根据提示输入许可证密钥，格式为 XXXX-XXXX-XXXX-XXXX-XXXX
  步骤4：按回车键，程序会自动连接授权服务器验证密钥
  步骤5：验证成功后弹出提示框，显示授权到期时间

  注意：
    - 许可证密钥需要购买获得
    - 密钥绑定到当前电脑，更换电脑需要重新绑定
    - 输入错误的密钥会自动清除，不会保存

------------------------------------------------------------------------
5.5 查看授权状态 CheckLicense
------------------------------------------------------------------------

  用途：查看当前授权状态、机器指纹等信息。

  步骤1：在命令行输入: CheckLicense
  步骤2：按回车键，命令行显示：
         - 当前授权状态（试用/已激活）
         - 机器指纹（用于绑定）
         - 当前密钥状态

七、常见问题解答
================================================================================

  Q1：输入命令后提示"未知命令"？
  A1：说明插件未加载成功。请重新执行 NETLOAD 命令加载 CadTrans.dll。

  Q2：翻译时CAD卡住不动？
  A2：翻译需要联网访问翻译服务器，请检查：
      - 电脑是否连接互联网
      - 公司网络是否屏蔽了外网访问
      - 翻译大量文字时需要等待，每个文字约1秒

  Q3：翻译显示"失败: 翻译结果为空"？
  A3：可能原因：
      - 当日免费额度已用完（匿名5,000字符/天）
      - 网络请求超时
      - 文本内容太长（单次最多500字符）
      解决：在设置窗口填写邮箱，额度提升至50,000字符/天

  Q4：翻译显示"失败: 创建文本失败"？
  A4：弹出错误对话框时，请将错误信息反馈给开发者。
      可能是文字样式创建失败，可尝试：
      - 在设置窗口切换翻译模式（新建↔替换）

  Q5：翻译后的中文显示为问号或方块？
  A5：系统缺少黑体字体(simhei.ttf)。
      - 安装黑体字体：将simhei.ttf复制到 C:\Windows\Fonts\

  Q6：如何取消正在执行的翻译？
  A6：按键盘左上角的 Esc 键可以中断操作。

  Q7：翻译后如何撤销？
  A7：按 Ctrl+Z 撤销上一步操作。

  Q8：每次打开AutoCAD都要重新加载插件吗？
  A8：默认是的。请参考"二、安装插件"中的方式B设置自动加载。

  Q9：选择文字时选不上？
  A9：请确认选择的是文字对象（鼠标悬停时会高亮显示），
      图块内的文字无法直接选中，需要先分解图块。

  Q10：支持哪些AutoCAD版本？
  A10：当前版本基于AutoCAD 2020开发，兼容2020-2024版本。
